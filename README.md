# Новогодишни късмети (APK без PC чрез GitHub Actions)

Този проект е готов за качване в GitHub и автоматично build-ване на APK.

## Как (само от Android телефон)
1) Инсталирай GitHub app (или отвори github.com) и влез/направи акаунт.
2) Направи ново repository (Public или Private).
3) Разархивирай този ZIP на телефона.
4) В GitHub repo-то: Add file -> Upload files -> качи всички файлове/папки от разархивираната папка.
5) После: Actions -> Build APK -> Run workflow.
6) Отвори run-а и свали Artifacts -> app-debug -> APK.

## Инсталиране на APK
- Отвори сваления APK и разреши “Install unknown apps”, ако поиска.
